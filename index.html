  • Some phonetically similar Nepali letters:
    ta = त, Ta = ट, tha = थ, Tha = ठ, da = द, Da = ड, dha = ध, Dha = ढ, na = न, Na = ण, sha = श, shha = ष
  • Some other special characters:
    • rr= र्‍ (as in garryo = गर्‍यो)
    • chha = छ
    • i = इ (as in miti = मिति)
    • ii = ई (as in nepaalii = नेपाली)
    • ^ = अनुस्वर (ं )
    • * = चन्द्रबिन्दु (ँ )
